# 📊 Swiggy Sales Dashboard (Excel Project)

## 🔹 Overview

This project analyzes Swiggy food delivery data to track business performance using an interactive Excel dashboard.

---

## 🔹 Key Metrics

* Total Orders
* Net Revenue
* Average Order Value
* Total Discount
* Cancellation Rate
* Average Rating

---

## 🔹 Features

* Dynamic Dashboard with KPIs
* Pivot Tables & Slicers (Month, City, Channel, Cuisine)
* City-wise and Monthly Performance Analysis
* Top Cuisines & Restaurants Insights

---

## 🔹 Tools Used

* Microsoft Excel
* Pivot Tables
* Data Cleaning
* Dashboard Design

---

## 🔹 Insights

* Mumbai generated highest revenue
* South Indian & Biryani are top cuisines
* Cancellation rate is low (~1%)
* Average rating is ~3.9
